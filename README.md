# Expense Tracker (Kuluseuranta)

## Kuvaus
Expense Tracker on Pythonilla tehty ohjelma henkilökohtaisten menojen seuraamiseen. Menot tallennetaan CSV-tiedostoon ja niitä voi lisätä ja tarkastella helposti.

## Ominaisuudet
- Lisää uusia kuluja päivämäärän, kategorian ja summan kanssa
- Näytä kaikki kulut
- Tallenna kulut automaattisesti CSV-tiedostoon

## Vaatimukset
- Python 3.x
- csv-kirjasto (Pythonin sisäänrakennettu)

## Käyttö
1. Lataa koodi GitHubista tai kloonaa repository
2. Siirry projekin kansioon
3. Suorita ohjelma:
   ```bash
   python expense_tracker.py
